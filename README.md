# 🚀 Samir Portfolio Website

## 🌐 Overview

A modern personal portfolio website built using React (Vite) and
Tailwind CSS.

It showcases my projects, skills, and development journey with a focus
on: - Clean UI/UX - Smooth animations - Scalable architecture

------------------------------------------------------------------------

## 🛠️ Tech Stack

-   **Frontend:** React (Vite), Tailwind CSS\
-   **Animations:** Framer Motion\
-   **Icons:** React Icons

------------------------------------------------------------------------

### 🔹 Scalable Architecture

-   Organized folder structure (sections, components, hooks, lib,

-   Built with reusability and scalability in mind

-   Clean 3-part layout (Logo, Navigation, Social Links)
